# Directorio de Empresas de Calatayud

Directorio con **1.397 entradas** de empresas, sociedades, comunidades de propietarios, asociaciones y otras entidades registradas en Calatayud (Zaragoza).

## Distribución por facturación (Axesor)

| Facturación | Empresas |
|---|---|
| > 100M € | 1 |
| 25M–100M € | 2 |
| 5M–25M € | 13 |
| 500K–5M € | 126 |
| < 500K € | 442 |
| **Total** | **584** |

*Nota: las 584 empresas con datos de facturación son un subconjunto de las 1.397 entidades del listado completo (se excluyen comunidades de propietarios, asociaciones, UTEs, SATs, etc.)*

## Fuentes

CSV con delimitador `;` y codificación UTF-8 con BOM (compatible con Excel).
